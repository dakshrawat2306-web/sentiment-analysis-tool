# sentiment-analysis-tool
NLP web app that analyzes product reviews and classifies  them as Positive, Negative, or Neutral using NLTK's VADER  sentiment analysis. Features a complete text preprocessing  pipeline, Flask REST API backend, and an interactive web UI  with real-time confidence scores. Deployed live on Render.
Sentiment Analysis Web App | Python · NLTK · VADER · Flask

• Built an end-to-end sentiment analysis web application that classifies product reviews as Positive, Negative, or Neutral using NLTK's VADER (Valence Aware Dictionary and sEntiment Reasoner).
• Implemented a complete NLP preprocessing pipeline including lowercasing, URL removal, punctuation cleaning, tokenization, and stopword removal.
• Leveraged VADER's compound scoring system to produce confidence-based sentiment predictions optimized for short-form and informal text.
• Developed a REST API backend using Flask to expose the model as a live /predict endpoint accepting JSON requests.
• Built an interactive web UI allowing users to input any review and receive real-time sentiment results with confidence scores.
• Deployed the application on Render, making it publicly accessible via a shareable live URL.
